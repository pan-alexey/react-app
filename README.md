# REACT EXAMPLE

### Implemented
* React Server Side Rendering + Redux
* Hot module replacement
* Typescript
* ESLint + Pretier
* CSS Modules

#### Run development mode
1. ```npm install```
2. ```npm run start```

#### Run lint
1. ```npm run lint``` - only lint
2. ```npm run lint:fix``` - lint & fix

#### JEST unit tests
```npm run test```
